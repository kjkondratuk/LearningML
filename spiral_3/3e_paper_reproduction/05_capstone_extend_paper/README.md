# Module 05: Capstone -- Extend a Paper

## Project
Choose one of the three reproduced papers and design an original extension.

## Requirements
1. 1-page research proposal (use `proposal.md` template)
2. Implement the extension
3. Run experiments with proper baselines and ablations
4. Statistical analysis (confidence intervals, significance tests)
5. 4-page report in conference format (use `report.md` template)

## Suggested Extensions
- **Transformers:** Mixture of Experts layers; study scaling behavior
- **DDPM:** v-prediction vs epsilon-prediction; few-step distillation
- **DPO:** Compare DPO, IPO, and KTO on the same dataset; study preference data quality
