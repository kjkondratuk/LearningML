# Capstone: CV Project -- DETR Object Detector

## Requirements
1. ResNet-50 backbone feature extractor
2. Transformer encoder-decoder
3. Bipartite matching loss (Hungarian algorithm)
4. Train on PASCAL VOC or subset of COCO
5. Evaluate with mAP > 30 on VOC
6. Visualize detections
7. Compare against YOLO baseline

## Reference
- Carion et al. 2020 (arXiv:2005.12872)
