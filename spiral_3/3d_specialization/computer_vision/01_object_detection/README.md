# Module: Object Detection

## References: Redmon et al. 2015 YOLO (arXiv:1506.02640), Carion et al. 2020 DETR (arXiv:2005.12872).

## Exercises
1. `compute_iou` -- Intersection over Union for bounding boxes.
2. `non_max_suppression` -- NMS to remove duplicate detections.
3. `anchor_box_generation` -- Generate anchor boxes.
4. `yolo_loss` -- YOLO v1 loss function.
5. `detr_bipartite_matching` -- Hungarian matching for DETR.
