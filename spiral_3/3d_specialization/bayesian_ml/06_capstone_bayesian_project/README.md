# Capstone: Bayesian Project -- Medical Uncertainty Quantification

## Project
BNN for medical uncertainty on Skin Cancer MNIST or similar.

## Requirements
1. MC Dropout and Bayes by Backprop implementations
2. Deep ensembles baseline
3. Evaluate: accuracy, calibration (ECE), OOD detection (AUROC)
4. Demonstrate "I don't know" on OOD examples
5. Compare uncertainty quality across methods
6. Reliability diagrams
7. 2-page report
