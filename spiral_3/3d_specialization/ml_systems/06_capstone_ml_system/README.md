# Capstone: ML System

## Project
Build a complete serving system for a 7B LLM.

## Requirements
1. INT4 quantization (GPTQ-style)
2. KV cache with PagedAttention concepts
3. REST API with continuous batching
4. Speculative decoding with draft model
5. Benchmark: tokens/sec, TTFT, throughput
6. Profile and identify bottlenecks
7. Compare against vLLM/TGI
8. 2-page report
