# Module 07: Capstone -- RL Agent

## Project
Build a PPO agent for continuous control (Gymnasium HalfCheetah or Ant).

## Requirements
1. PPO with all modern tricks (GAE, clipping, entropy bonus, observation normalization)
2. Model-based component for data augmentation (Dyna-style)
3. Compare model-free PPO vs model-augmented PPO
4. Learning curves with confidence intervals (5 random seeds)
5. CQL comparison on offline dataset from trained agent
6. 2-page report
