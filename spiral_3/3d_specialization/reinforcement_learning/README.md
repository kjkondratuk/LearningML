# Reinforcement Learning Track

## Overview

This track covers reinforcement learning from foundations (MDPs, dynamic programming)
through modern algorithms (PPO, model-based RL, offline RL). Each module implements
algorithms from key papers and verifies convergence and optimality properties.

## Prerequisites

- Probability and statistics (Markov chains, expectations)
- Optimization (gradient descent, convergence)
- Spiral 2 deep learning basics
