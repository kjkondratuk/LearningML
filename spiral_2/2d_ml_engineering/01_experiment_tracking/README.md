# Module 01: Experiment Tracking

> Running 10 experiments and saving results in print statements is a recipe for
> lost work. This module introduces MLflow for structured experiment tracking.

## "Naive then Derive" Challenge

Run 10 training experiments with different hyperparameters, saving results in text
files. Try to figure out which was best. Then implement MLflow tracking and show how
comparison becomes trivial.
